How template engine works.

Django uses its own template system called the Django Template Language.

Rendering process:

  1. A view calls render(request, template_name, context).
  2. Django loads the template file and parses it into tokens.
  3. The engine replaces variables and executes template tags.
  4. Produces a final HTML string.
  5. Returns it as an HTTP response.

Blocks

Blocks are used with template inheritance and it allows child templates to override parts of a base template.
